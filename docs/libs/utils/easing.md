# snu.easeInSine() 

### **缓动动画 - 正弦曲线入**

使用正弦曲线作为缓动动画的入（开始）效果。

### 兼容性

| Web  | Android |
| ---- | ------- |
| √    | √       |

### 属性

| 名称 | 类型   | 必填 | 默认值 | 描述                      |
| ---- | ------ | ---- | ------ | ------------------------- |
| t    | Number | 是   | -      | 当前时间（0 到 1 之间）。 |

### 返回值

| 类型   | 必备 | 描述                       |
| ------ | ---- | -------------------------- |
| Number | 是   | 返回当前时间对应的动画值。 |

### 示例
``` typescript
const value = snu.easeInSine(0.5);
```

| t    | Number | 是   | -      | 当前时间（0 到 1 之间）。 |

### 返回值

| 类型   | 必备 | 描述                       |
| ------ | ---- | -------------------------- |
| Number | 是   | 返回当前时间对应的动画值。 |

### 示例
``` typescript
const value = snu.linear(0.5);
```
