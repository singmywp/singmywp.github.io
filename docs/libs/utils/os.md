# snu.vibrate()

### **振动设备**

使振动器产生振动。

### 兼容性

| Web  | Android |
| ---- | ------- |
| ×    | √       |

### 属性

| 名称 | 类型   | 必填 | 默认值 | 描述                   |
| ---- | ------ | ---- | ------ | ---------------------- |
| time | Number | 否   | 200    | 振动持续时间（毫秒）。 |

### 返回值

无
