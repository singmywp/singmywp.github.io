# snu.isDate()

### **是否为日期**

验证输入的值是否为有效的日期。

### 兼容性

| Web  | Android |
| ---- | ------- |
| √    | √       |

### 属性

| 名称 | 类型 | 必填 | 默认值 | 描述     |
| ---- | ---- | ---- | ------ | -------- |
| date | Any  | 是   | -      | 输入的值 |

### 返回值

| 类型    | 必备 | 描述                      |
| ------- | ---- | ------------------------- |
| Boolean | 是   | 如果是有效日期，返回 true |

---
