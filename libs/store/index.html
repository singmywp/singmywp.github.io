Store 状态管理 ​

一个轻量简洁的全局响应式状态管理库。支持持久化数据储存、过期机制。

配置方法如下：

typescript
import { $snui } from '@/uni_modules/sinle-ui'
const store = $snui.store

接下来，本文档均以 store 代指 $snui.store。
